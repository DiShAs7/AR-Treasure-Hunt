# 🪙 AR Treasure Hunt

A fun Augmented Reality coin collection game built with Unity and ARCore.

## Features
- Coins spawn on real-world flat surfaces
- Tap coins to collect them
- Bonus purple coins worth 3x points
- 60 second countdown timer
- Score system with final score screen
- Sound effects on collection

## Built With
- Unity 2022.3
- AR Foundation
- ARCore (Android)

## How to Play
1. Point your camera at a flat surface (floor/table)
2. Wait for coins to appear
3. Walk to coins and tap them to collect
4. Collect as many as possible before time runs out!
